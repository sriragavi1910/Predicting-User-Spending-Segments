
# Predicting User Spending Segments in Mobile Game Data

This project focuses on predicting user spending behavior in mobile games by segmenting users into different spending categories using machine learning techniques.

## Problem Statement
Mobile gaming companies need to identify high-value and low-value users to improve monetization strategies and personalized offers.

## Solution
A machine learning model was developed to analyze user behavior data and classify users into spending segments.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Machine Learning

## Project Objective
To predict user spending segments and help gaming businesses improve decision-making and revenue strategies.
